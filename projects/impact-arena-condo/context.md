## 6. Operation ปัจจุบัน

**Check-in Process (Self Check-in):**
1. ลูกค้าจองผ่าน Platform
2. แม่ทักในแชท Platform → ให้แอด Line ส่วนตัว + เบอร์โทร
3. ส่งรายละเอียดห้อง: ตึก, ห้อง, กุญแจวางบนตู้เย็น, ประตูไม่ล็อก
4. เข้าตึก: รอให้ลูกบ้านเปิดประตูแล้วตามเข้า
5. Check-out: ปิดแอร์/ไฟ, วางกุญแจในห้อง, ไม่ต้องล็อก

**ปัญหาระบบแสกนหน้า:**
- C2: แสกนหน้าบังคับ → ต้องพาลูกค้าไปลงทะเบียนกับนิติฯ เสียเวลา ~10+ นาที → Bond ไม่นำ C2 ลงบัญชีตัวเอง
- C8: ใช้คีย์การ์ดแทนได้ → แก้โดยรับลูกค้าหน้าคอนโดหรือรอลูกบ้านเปิด
- **อนาคต:** แสกนหน้าจะขยายไปทุกตึก — เพิ่มภาระ Bond/แม่ต้องพาลูกค้าลงทะเบียนทุกครั้ง

**ทำความสะอาด:**
- แม่ + Bond ช่วยกัน บางครั้งจ้างเพิ่มเมื่อลูกค้าเยอะ (ตอนนี้ลูกค้าน้อยเกินไป ต้นทุนจ้างไม่คุ้ม)
- ผ้าปู: ซักทุก 3 วันการจอง (ไม่ใช่ทุกครั้งที่ลูกค้าออก)
- ผ้าขนหนู: ซักทุกครั้ง
- แม่ซักผ้าที่บ้าน — เอาขึ้นรถกลับเย็น นำมาคืนเช้า (กระทบคนในบ้าน แก้ไม่ได้เพราะไม่มีงบโกดัง)
