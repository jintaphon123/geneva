# Second Brain — Project Context

## Status
**Active — Primary Focus**

## What It Is
Platform-agnostic AI memory system using `.md` files + skill folders. Context persists and compounds across any LLM (Claude, ChatGPT, Gemini, etc.). Knowledge grows with every session.

## Key Differentiator
- Backend is `.md` files — no vendor lock-in, no proprietary format
- Works across all LLMs by loading context files at session start
- Designed for non-tech users: ChatGPT-like UI hides all complexity

## Target User
Non-tech-savvy people who want AI that remembers them and compounds knowledge over time.

## Stack
