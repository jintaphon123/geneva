---
name: geneva-handoff
description: Package current session context for seamless handoff to next session or agent
---

When invoked at session end:
1. Summarize what was worked on (3-5 bullets)
2. List decisions made with rationale
3. List open items and blockers
4. Identify context that must be loaded next session
5. Propose memory updates to save

Output: structured handoff block ready to paste into MEMORY.md or next session prompt.
