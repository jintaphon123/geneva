---
name: scenario-planner
description: Build structured best/base/worst case scenarios for decisions or projects
---

When invoked with a decision or project context:

1. Best case: what does success look like in 3mo / 1yr?
2. Base case: realistic outcome given current resources and constraints
3. Worst case: what is the specific failure mode and its probability?
4. Trigger indicators: what signals would tell you which scenario is unfolding?
5. Hedges: what can be done now to reduce worst-case probability?

Output as structured table + 1-paragraph recommendation.
