---
name: self-improvement-review
description: Audit this session for gaps in speed, accuracy, and system quality — propose fixes
---

When invoked:
1. Scan conversation for: re-derived context, repeated errors, slow tasks, vague answers
2. For each gap: what was slow/wrong? what triggered it? what would fix it?
3. Propose: new skill, new SOP, context update, or rule change
4. Output structured entries ready for references/capability-gaps.md

Self-assessment questions:
- What would have made this session faster?
- What context should have been loaded but wasn't?
- What rule was violated or missing?
- What skill would have made this instant instead of manual?
